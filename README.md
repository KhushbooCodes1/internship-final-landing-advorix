# 🚀 Internship Program Landing Page

A modern, responsive landing page for an internship program built using *HTML, CSS, and JavaScript*.  
Designed with a clean UI, smooth animations, and fully responsive layout.

---

## 📌 Features

- ✅ Modern UI with gradient design (dark blue + purple)
- ✅ Fully responsive (Mobile, Tablet, Desktop)
- ✅ Smooth scrolling navigation
- ✅ Animated scroll reveal (Intersection Observer)
- ✅ Interactive FAQ section
- ✅ Form validation with error messages
- ✅ Responsive hamburger menu
- ✅ Clean and reusable code structure
